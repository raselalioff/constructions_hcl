Human Resources - HR Employee Expense Advance Request Odoo Apps

Human Resource Employee Expense Advance Request Odoo App allows your employees to create advance request for expenses. This app will also work with multi currency. Once Employee Expense Advance Request is submitted it will ask for approval from the Human Resource -HR manager and manager can approve and pay the employee HR advance expense request.

Features

Maintains advance expense request

App allows the user to drop a request about expenses.

Allows manager to pay all expenses.

Allows user to get paid with all expenses and journel gets created.

Offers a request of approval to manager

User needs to fill up all necessary details about expense and asks for approval to manager.

Prints report about the all the expense.

Allows user to print a report about all the expenses paid and unpaid.

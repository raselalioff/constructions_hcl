Equipment Request & IT Operations Odoo Apps

If you are looking for an app for managing the employee request for equipments,this app can really be helpful. This app allow employees to send requests to HR department for equipments type of hardware and software.

Features

Allows user to request for equipment

App offers user to apply for both hardware and software equipment.

Integrated with warehouse.

Allows HR manager to create an internal transfer.

Allows user to get multiple validation from different deparments.

Allows user to request for multiple equipment to different authorities.

Allows to print a report.

Allows to print a report of equipment request.
Demo Request Video Tutorial

Create an Equipment Request

Creates an equipment request for the employee.


Create damage details for the product.

Reject Equipment Request

Reject equipment request by clicking on 'Reject' button.


User can see rejected equipment request and also can reset to draft request.

Apply for Department Approval

After getting confirm by user can apply for department approval.

Refused Equipment Request

Refused equipment request by clicking on 'Refused' button.


User can see refused equipment request and also can reset to draft request.

Apply for HR Approval

After getting department approval user can apply for HR approval.


When user can apply for HR approval without generate expense, Confirmation popup will show.

Generate Expense

When click on 'Generate Expense' button expense will be generated and show in 'Expenses' tab.


Creates an Internal Transfer

After getting HR approval user can create an internal transfer for form by HR with selection of source location.

View Internal Transfer


User can view an internal transfer request which goes in warehouse directly.

Print Equipment Request

User can easily print equipment request in PDF format.
